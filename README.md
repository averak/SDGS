This app is documents generator for RCC.

## Requirement

- Python 3.9

## Develop

### Installation

```bash
$ pipenv install
```

### Code check & format

```bash
# code check
$ pipenv run lint && pipenv run mypy

# code format
$ pipenv run format
```

## Usage

You can run this application with pipenv.

```bash
$ pipenv run start
```
